## WeChat mini program - Rubik's Cube

基于 ThreeJS 框架的魔方微信小游戏
ThreeJS/微信小程序/JavaScript 

2023.8.11成功本地运行并提交了微信审核

## 源码目录介绍
```
./adapter                                  // 适配库
./images                                   // 素材图片
./game.js                                  // 入口脚本
./game.json                                // 配置文件
./js
├── object
│   ├── Rubik.js                           // 简易魔方对象
│   └── TouchLine.js                       // 滑动条对象
├── threejs
│   └── three.js                           // threejs框架源代码
├── tween
│   └── tween.js                           // tweenjs框架源代码
└── main.js                                // 游戏主函数
```
